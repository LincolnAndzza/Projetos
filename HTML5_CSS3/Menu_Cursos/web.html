Programador Web

Curso que prepara o aluno a desenvolver sites e sistemas para web,especificando,testando,realizando
manutenção e documentando suas funcionalidades de acordo com as melhores práticas
de mercado.

Sobre o curso

Criar sites web dinâmicos;
Desenvolver layout e formatações responsivas para páginas/aplicações web;
Desenvolver soluções através de algoritmos, lógica e linguagens de programação;
Criar e manipular bases de dados utilizando SGBD’s do mercado;
Distribuir e manter páginas/aplicações web com segurança;
Desenvolver um sistema e-Commerce de acordo com os requisitos de segurança apropriados
